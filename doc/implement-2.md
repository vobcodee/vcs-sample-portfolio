# 2단계 상세 구현 명세서

## 목업 디자인 반영 (About 섹션 구현)

---

## 1. 단계 목표 요약

- 제공된 **About 목업 디자인을 코드로 정확히 구현**
    
- 디자인 → 컴포넌트 구조 전환 검증
    
- 이후 확장을 고려한 **구조적 구현**
    

---

## 2. 2단계 구현 범위

### In Scope

- About 섹션 UI 구현
    
- 레이아웃 / 타이포그래피 / 컬러 반영
    
- 정적 데이터 기반 렌더링
    
- Desktop 기준 완성
    

### Out of Scope

❌ 모바일 세부 최적화  
❌ 인터랙션 애니메이션  
❌ SEO 설정  
❌ 실제 프로젝트 데이터 연동

---

## 3. 구현 대상 화면 구성

```
About Section
 ├ Profile Image Area
 ├ About Text Content
 ├ Info Grid (4 Blocks)
 └ Resume CTA Card
```

---

## 4. 상세 To-Do 리스트

---

## 4.1 About 섹션 컴포넌트 생성

-  `AboutSection.tsx` 생성
    
-  메인 페이지(`page.tsx`)에 섹션 연결
    
-  섹션 wrapper 스타일 정의
    

**완료 기준**

- About 섹션이 독립 컴포넌트로 분리됨
    

---

## 4.2 레이아웃 구현 (2-Column)

-  CSS Grid 또는 Flex 적용
    
-  좌/우 2-Column 구조 구현
    
-  최대 콘텐츠 너비 제한
    
-  좌우 간격(gap) 설정
    

**완료 기준**

- 이미지 영역과 텍스트 영역 명확히 분리
    

---

## 4.3 프로필 이미지 영역 구현

-  이미지 컴포넌트 추가
    
-  border-radius 적용
    
-  흑백 이미지 스타일 적용
    
-  이미지 하단 위치 정보 영역 구성
    

**세부 항목**

- 위치 아이콘 + 텍스트
    
- 이메일 / 링크 아이콘 버튼
    

**완료 기준**

- 목업과 동일한 이미지 카드 형태
    

---

## 4.4 섹션 라벨 (ABOUT ME)

-  캡슐 형태 라벨 구현
    
-  대문자 텍스트
    
-  Primary 컬러 연한 배경 적용
    
-  라벨 위치 정렬
    

**완료 기준**

- 섹션 시작을 명확히 인지 가능
    

---

## 4.5 메인 헤드라인 구현

-  큰 폰트 사이즈 적용
    
-  줄바꿈 구조 구현
    
-  이름 텍스트 강조 컬러 적용
    

**완료 기준**

- 시각적 위계가 명확한 헤드라인
    

---

## 4.6 설명 텍스트(Description) 구현

-  2개 문단 텍스트 작성
    
-  line-height 조정
    
-  최대 줄 길이 제한
    

**완료 기준**

- 읽기 편한 본문 영역 완성
    

---

## 4.7 정보 그리드(Info Grid) 구현

-  2x2 Grid 레이아웃 구성
    
-  `InfoItem` 컴포넌트 생성
    
-  아이콘 + 제목 + 텍스트 구조
    

### 포함 항목

- 관심 분야
    
- 작업 스타일
    
- 주요 도구
    
- 학력
    

**완료 기준**

- 정보가 균형 있게 정렬됨
    

---

## 4.8 Resume CTA 카드 구현

-  카드 형태 컨테이너 구현
    
-  좌측 텍스트 영역 구성
    
-  우측 CTA 버튼 구현
    
-  버튼 hover 상태 정의
    

**완료 기준**

- 클릭 유도 CTA 명확히 보임
    

---

## 4.9 스타일 정리 (CSS Modules)

-  About 전용 CSS Module 생성
    
-  공통 색상 변수 사용
    
-  불필요한 중복 스타일 제거
    

**완료 기준**

- About 섹션 스타일 캡슐화
    

---

## 4.10 반응형 최소 대응 (선택)

-  태블릿 기준 column 전환
    
-  텍스트 overflow 점검
    

**완료 기준**

- 화면 깨짐 없음
    

---
